PImage

PImage on muuttuja tyyppi, johon tallennetaan kuvia Processing-ohjelmissa.

loadImage("kuvan_osoite")

loadImage() lataa kuvan verkosta tai tiedostosta tietokoneelta. Jos kuva on tallennettu ohjelman data-kansioon riittää, että annetaan ainoastaan kuvatiedoston nimi. Tämä metodi tulee ajaa setup-metodin sisällä ja saattaa kestää hetken jos kuva on iso.

image(kuva, x, y)

image() piirtää annetun kuvan haluttuihin koordinaatteihin.

Esim. seuraava ohjelma lataa verkosta linnun kuvan ja piirtää sitä sitten kursorin osoittamaan paikkaan. 
// esitellään linnun_kuva muuttuja
PImage linnun_kuva;

void setup() {
  size(800, 600);
  // ladataan kuva verkosta
  linnun_kuva = loadImage("http://www.wpclipart.com/animals/birds/_Miscellaneous/generic_colors/clip_art_bird_blue.jpg");
}

// uudelleen ja uudellen ajettava koodi
void draw() {
  background(255);
  // piirtää kuvan kursorin kohdalle
  image(linnun_kuva, mouseX, mouseY);
}
